# Code of Conduct

> Status: Finalisiert fuer V1-Buildstart
> Stand: 2026-03-18
> Geltung: Verbindliche Arbeitsgrundlage


This project follows a professional, respectful collaboration standard.

## Expected Behavior
- Be respectful and constructive.
- Focus feedback on code, specs, and outcomes.
- Assume good intent and resolve disagreements with evidence.
- Escalate conflicts early and keep discussions factual.

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks.
- Sharing private information without permission.
- Repeated disruptive behavior.

## Enforcement
Project maintainers may remove content or restrict participation when behavior violates this policy.

## Reporting
Report conduct issues directly to repository maintainers via private channel.
