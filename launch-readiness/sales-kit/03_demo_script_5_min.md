# 5-Minuten Demo-Skript

## Minute 0:00-0:45 — Kontext

- "Tap'em ist eure gym-interne Mitglieder-App fuer Tracking, Community und Admin-Steuerung."
- "Fokus: alltagstauglicher Mehrwert fuer Mitglieder und Studio-Team."

## Minute 0:45-2:00 — Member-Flow

- Workout starten (NFC oder manuell)
- Sets tracken, Session abschliessen, Sync sichtbar
- Kurz zeigen: Community/Friends/Rankings

Key Message:
- "Mitglieder bekommen eine einfache Trainingsroutine statt Notizchaos."

## Minute 2:00-3:45 — Admin/Owner-Flow

- Owner Overview:
  - Aktivierungsfunnel
  - Nutzung
  - NFC-Abdeckung
  - offene Aufgaben
- Admin-Hub:
  - Join-Requests
  - Geraete-Feedback
  - Moderation
- Equipment Analytics:
  - Nutzung
  - NFC-Status
  - Feedback-Aufschluesselung

Key Message:
- "Ihr bekommt eine operativ nutzbare Sicht statt Bauchgefuehl."

## Minute 3:45-4:30 — Challenges

- Challenge-Templates zeigen
- KPI-Sicht erklaeren:
  - Teilnahme
  - Completion

Key Message:
- "Challenges werden steuerbar statt nur sichtbar."

## Minute 4:30-5:00 — Pilotabschluss

- Pilotdauer + KPI-Review Zyklus erklaeren
- Nächster Schritt:
  - "Kickoff-Termin + Setup-Checkliste + Pilotstartdatum fixieren"
