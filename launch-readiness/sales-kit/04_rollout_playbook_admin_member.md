# Rollout Playbook (Admin + Member)

## Ziel

Innerhalb der ersten 7 Tage ein neues Gym stabil live bringen.

## Day 0 — Vorbereitung

- Gym-Profil und Join-Code pruefen
- Equipment-Liste validieren
- NFC-Tagging Startset definieren
- Admin-Rollen fixieren
- Verantwortliche Person im Studio benennen

## Day 1 — Admin Kickoff

- 30-45 Minuten Admin-Session:
  - Admin Hub
  - Owner Overview
  - offene Aufgaben
  - Equipment-Feedback-Prozess
  - Moderationsprozess

## Day 2-3 — Member Aktivierung

- In-App QR/Code intern teilen
- Trainer-Team briefen (wie Workout-Start erklaeren)
- Erste Challenge aus Template starten

## Day 4-5 — Operativer Feinschliff

- Join-Requests SLA pruefen
- offene Feedbacks priorisieren
- NFC-Luecken reduzieren

## Day 6-7 — Erste KPI-Review

- Aktivierung (Join -> erstes Workout)
- Nutzung (Workouts/aktive User)
- Challenge (Teilnahme/Completion)
- Ops (offene Tasks)

## Mindeststandards fuer "stabil live"

- Keine kritischen Blocker im Workout-Flow
- Admin-Task-Backlog unter definiertem Schwellwert
- Mindestens eine aktive Challenge
- Verantwortliche Person im Studio kann Hub + Owner Overview selbst bedienen
