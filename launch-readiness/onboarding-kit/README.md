# Gym Onboarding Kit (Launch-Ready)

Stand: 2026-04-03
Zweck: Ein neues Gym ohne Improvisation reproduzierbar in 1-2 Tagen live bringen.

## Enthaltene Artefakte

1. `01_gym_setup_checklist.md`
2. `02_equipment_intake_template.md`
3. `03_nfc_tagging_checklist.md`
4. `04_admin_onboarding_sheet.md`
5. `05_pilot_rollout_day_checklist.md`
6. `06_new_gym_dry_run_protocol.md`
7. `07_new_gym_dry_run_evidence_template.md`

## Reihenfolge (verbindlich)

1. Gym Setup Checklist ausfuellen und abnehmen.
2. Equipment Intake Template vom Studio uebernehmen und validieren.
3. NFC Tagging Checklist auf der Flaeche ausfuehren.
4. Admin Onboarding Sheet im Kickoff-Termin durchgehen.
5. Pilot Rollout Day Checklist am Go-Live-Tag ausfuehren.
6. Dry-Run Protocol einmal komplett durchlaufen.
7. Dry-Run Evidence dokumentieren und abzeichnen.

## Definition of Done fuer Onboarding

Onboarding ist erst abgeschlossen, wenn alle Punkte erfuellt sind:

- Alle Pflichtfelder in Artefakten 1-5 sind ausgefuellt.
- Dry-Run gemaess Artefakt 6 erfolgreich bestanden.
- Evidence aus Artefakt 7 liegt vor.
- Studio-Verantwortliche Person hat schriftlich abgenommen.
- Keine P0/P1 Blocker im Workout-, Join- oder Admin-Flow offen.

## Rollen

- Tap'em Owner: Prozessfuehrung, technische Abnahme.
- Studio Owner/Manager: Freigaben, Ressourcen, finale Abnahme.
- Studio Admin: Tagesbetrieb in App (Members, Equipment, Challenges).
- Trainer Champion: Member-Einfuehrung auf der Flaeche.
