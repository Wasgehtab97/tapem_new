# 04 - Admin Onboarding Sheet

Stand: 2026-04-03
Zweck: Sicherstellen, dass das Studio den Betrieb ohne dich steuern kann.

## Terminrahmen

- Empfohlene Dauer: 45 Minuten
- Teilnehmer: Studio Owner/Manager + 1-2 Admins + Trainer Champion

## Agenda (verbindlich)

1. Rollen und Verantwortlichkeiten (5 min)
2. Admin Hub Navigation (10 min)
3. Members + Join Requests Prozess (10 min)
4. Equipment/NFC Pflegeprozess (10 min)
5. Challenges starten + KPI lesen (5 min)
6. Eskalationsweg bei Problemen (5 min)
